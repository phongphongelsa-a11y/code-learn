# OPERATORS # 
## Toán tử gán, các phép so sánh ##
| Phép tính | Công thức | Ý nghĩa |
| :---  | :---: | ---: |
| = ( = là gán ) | x=1 | x=1 | 
| += | x+=y | x=x+y |
| -= | x-=y | x=x-y | 
| *= | x*=y | x=x*y |
| += | x+=y | x=x+y |
| /= ( này là chia lấy phần nguyên )  | x/=y | x=x/y | 
| %= ( này là chia lấy phần dư ) | x%=y | x=x%y | 


## so sánh ##
| Phép tính | Công thức | Ý nghĩa  | Nội dung |
| :--- | :---: | :---: | ---: |
| <=  | < hoặc =  | 2<=2 | True |
| >=  | > hoặc =  | 2>=1 | True |
| ==  | = nhau  | 1==1 | True |
| !=  | khác nhau  | 1!=1 | False |
| >  | lớn hơn  | 1>2 | False |
| <  | nhỏ hơn  | 2<1 | False |
