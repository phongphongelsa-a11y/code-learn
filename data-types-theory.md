# CÁC KIỂU DỮ LIỆU CƠ BẢN TRONG C++ (DATA TYPES) #

| Kiểu dữ liệu | Ý nghĩa | Kích thước | Khoảng giá trị (Tương đối) |
| :--- | :--- | :---: | :--- | :--- |
| `int` | Số nguyên | 4 Bytes | Khoảng -2 tỷ đến 2 tỷ | 
| `long long` | Số nguyên siêu to | 8 Bytes | Rất lớn (~9 tỷ tỷ) | 
| `float` | Số thực (nhỏ) | 4 Bytes | ~7 chữ số thập phân | 
| `double` | Số thực (lớn) | 8 Bytes | ~15 chữ số thập phân |
| `char` | Ký tự đơn | 1 Byte | 256 ký tự (Bảng mã ASCII) | 
| `bool` | Logic Đúng/Sai | 1 Byte | `true` (1) hoặc `false` (0) |
